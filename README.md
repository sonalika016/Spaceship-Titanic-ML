# 🚀 Spaceship Titanic ML Project 🌌

### 🧠 Predicting Passenger Transportation Using Machine Learning

This project predicts whether passengers were transported to another dimension during the Spaceship Titanic incident.

---

## 📌 Models Used

* 🌲 Random Forest
* ⚡ XGBoost
* 🐱 CatBoost

---

## 🛠️ Feature Engineering

* 🆔 PassengerId Splitting
* 🚪 Cabin Splitting

---

## 📊 Accuracy

| Model            | Accuracy |
| ---------------- | -------- |
| 🌲 Random Forest | 74.58%   |
| ⚡ XGBoost        | 78.89%   |
| 🐱 CatBoost      | 78.83%   |
